## Purpose

- A simple budget app where you can add expenses and income to your current month, or for months in advance! Just create an account and add your bills!

## Technologies

- MERN
- React.js
- Node.js
- Express.js
- Express-Session
- MySQL
- Sequelize
- Styled Components
- JawsDB
- dotenv
- bcrypt

## Features

- Clean user interface
- Easy to use website
- Password encryption
- Add and delete bills and income
